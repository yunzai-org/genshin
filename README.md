## genshin

该插件完全依赖于喵喵插件

请先安装喵喵插件后再安装

```sh
# 喵喵插件
git clone --depth=1 https://gitee.com/yoimiya-kokomi/miao-plugin.git ./plugins/miao-plugin/
```

```sh
# 原神插件
git clone --depth=1 https://github.com/yunzai-org/genshin.git ./plugins/genshin/
```
